  # Parent–Teacher Communication Classification using BERT
  
  ## Project Overview
  
  This project fine-tunes Google's BERT transformer model to automatically classify parent–teacher communication messages into predefined categories. The objective is to assist educational institutions in organizing and analyzing communication efficiently.
  
  ---
  
  ## Problem Statement
  
  Schools receive hundreds of parent messages every day.
  
  Manual categorization is time-consuming.
  
  This project uses Natural Language Processing (NLP) to automate the classification process using BERT.
