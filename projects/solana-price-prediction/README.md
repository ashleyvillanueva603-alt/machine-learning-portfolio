# Solana Cryptocurrency Price Prediction

## Project Overview

This project explores cryptocurrency price prediction using historical Solana (SOL) market data and machine learning techniques. The objective is to analyze market trends and generate predictions based on historical price movements.

---

## Problem Statement

Cryptocurrency prices are highly volatile and influenced by various market factors.

This project applies machine learning techniques to historical Solana price data to better understand price behavior and evaluate predictive performance.
