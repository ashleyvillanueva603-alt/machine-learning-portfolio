# Named Entity Recognition using spaCy

## Project Overview

This project demonstrates Named Entity Recognition (NER) using the spaCy NLP library to automatically identify and classify named entities from natural language text. The system extracts meaningful information such as people, organizations, locations, dates, products, and other entities from unstructured text.

---

## Problem Statement

Extracting important information from textual documents manually is time-consuming and inefficient.

Named Entity Recognition automates the identification of meaningful entities, making it useful for document analysis, chatbots, search engines, healthcare, finance, and many other NLP applications.

---

## Dataset

- Sample English text corpus
- spaCy pre-trained English language model (`en_core_web_sm`)
